# CyberArmor 🛡️

An Security scanner built with Python and PyQt6.

## Features
- 🔍 Port Scanner
- 🔑 Password Manager & Strength Checker
- 📧 Email Spam Checker
- 🤖 AI Chatbot
- 📡 WiFi Speed Test
- 🔗 Malicious Link Detector
- 📁 File Scanner

## Built With
- Python
- PyQt6
- SQLite
